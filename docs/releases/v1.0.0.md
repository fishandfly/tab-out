# Tab Out v1.0.0

Tab Out 1.0 正式把新标签页升级成一个可日常使用的本地工作台。

## 核心更新

- 新增工作台 Tab：`GTD`、`白板`、`笔记`、`结构图`
- GTD 区域支持四象限任务管理、步骤拆解、拖拽调整层级、番茄钟，以及日报 Markdown 导入 / 导出
- 新增 Excalidraw 白板：支持全屏、标题原地编辑、`.excalidraw` 导入 / 导出、自动保存、加载后自动居中
- 新增 Tiptap 笔记：支持图标工具条、表格、全屏、标题原地编辑、Markdown `.md` 导入 / 导出
- 新增 draw.io 结构图：支持全屏、标题原地编辑、`.drawio` 导入 / 导出，并可继续编辑导入文件
- 新增系统暗色模式跟随：GTD、白板、笔记、结构图统一跟随 macOS / Windows 系统主题

## 交互与界面调整

- 工作台 Tab 被移动到顶部鼓励语下方，更适合作为主操作区
- 移除了顶部搜索区，避免和工作台入口冲突
- 调整标题区视觉样式，保留鼓励语黑色阴影并与工作台中心线对齐
- 白板、笔记、结构图都支持标题双击原地编辑，导出时默认使用当前标题作为文件名
- draw.io 初始加载时默认关闭形状和格式相关侧边面板，进入更干净

## 标签管理与主工作台保留能力

- 保留原有 Tab Out 的标签按域名分组浏览
- 保留重复标签检测与一键清理
- 保留稍后处理清单
- 保留首页分组、关闭反馈动画、背景图轮播和鼓励语

## 数据与运行方式

- 所有 GTD 数据、白板、笔记、结构图和界面偏好均保存在本地 `chrome.storage.local`
- 无服务端、无账号系统、无远程业务依赖

## 发布包

- Chrome 扩展包：`gtd-new-tab-v1.0.0.zip`
