# Tab Out

**把新标签页变成你的 GTD 工作台。**

[English](./README.md)

Tab Out 是一个 Chrome 新标签页扩展。它把原本只做标签整理的新标签页，扩展成一个集标签管理、GTD 四象限、执行步骤拆解、番茄专注、Google 搜索、背景图轮播和心灵鸡汤于一体的个人工作空间。

没有独立服务器，没有账号系统。核心数据保存在本地，同时使用轻量的在线背景图源提升视觉体验。

## 功能特性

- **GTD 四象限**：基于“重要 / 紧急”管理今日任务，支持象限内原地添加、任务拖拽、任务勾选，以及日报 Markdown 导出
- **番茄钟**：在当前任务的执行步骤区域直接启动 25 分钟专注计时，支持单击暂停 / 继续、双击重置
- **动态背景图**：在大气、壮观、运动感的背景图之间自动轮换，提升新标签页的氛围感
- **Google 搜索**：顶部内置搜索框，打开新标签页即可直接搜索
- **心灵鸡汤**：顶部轮播鼓励语句，定时切换，保持页面积极、轻快的工作状态
- **标签管理**：按域名分组展示当前所有打开的标签页，快速总览当前浏览状态
- **首页分组**：把 Gmail、X、YouTube、LinkedIn、GitHub 等首页聚合成独立分组
- **一键关闭标签**：保留原项目的关闭动画与反馈体验
- **重复标签检测**：发现重复页面并快速清理
- **稍后处理清单**：在关闭标签前先存起来，避免遗漏
- **本地开发分组**：对 localhost 标签按端口号区分，更适合开发使用
- **纯扩展运行**：无需后端、无需 Node 服务、无需额外部署

## 为什么要增加这些特性

Chrome 是我每天使用频率最高的工具，所以我希望这个新标签页除了管理标签之外，还能进一步提升我的工作效率。既然我一天会无数次打开这个页面，那它就不该只是一个“标签中转站”，而应该成为我最常进入的工作台。

我现在最大的现实问题，是时间太碎片化。每天看起来很忙，但很容易抓不住真正的重点。所以我希望在这个高频使用的页面上，直接放进能帮助我聚焦的功能。GTD 四象限和番茄钟，就是在这个思路下加进来的。前者逼我每天明确“重要且紧急”的事情，后者让我进入一段真实可执行的专注时间，而不是一直停留在计划层面。

但我又不希望这个页面只有工具性，没有一点情绪氛围。纯功能页面太硬了，用久了容易疲。于是我加了动态背景图和心灵鸡汤。它们不是摆设，而是为了给自己一点情感价值和精神鼓励，让聚焦这件事更容易坚持下去。

搜索框我选择的是 Google，而不是百度。原因也很直接：我想尽量减少被新闻信息流打断的时间。打开新标签页之后，我希望自己是“搜索一个答案然后继续工作”，而不是被别的内容一路带走。

另外，我对 GTD 的规则比较残酷：如果今天计划的事情没有做完，第二天不会自动继承，而是直接清空。如果它真的还重要，那我就重新录入。因为我自己很讨厌重复录入，这反过来会逼着我尽量把当天规划好的工作做完。

这些功能放在一起之后，这个新标签页对我来说就不只是标签页了，而是我现在最顺手、最直接、也最符合自己工作习惯的工作台。

## 手动安装

**1. 克隆仓库**

```bash
git clone https://github.com/fishandfly/tab-out.git
```

**2. 在 Chrome 中加载扩展**

1. 打开 `chrome://extensions`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择仓库中的 `extension/` 目录

**3. 打开一个新标签页**

你就会看到新的 GTD 工作台页面。

## 分享给同事

执行打包脚本：

```bash

```

同事安装方式：

1. 解压 zip
2. 打开 `chrome://extensions`
3. 打开“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择解压后的扩展目录

## 使用方式

```text
打开一个新标签页
  -> 顶部可以直接用 Google 搜索
  -> 上方会显示鼓励语句
  -> 中间区域是今日 GTD 四象限
  -> 点击任务后可在右侧拆解执行步骤
  -> 可直接启动番茄钟进入专注状态
  -> 下方继续保留原有的 Open Tabs 标签管理能力
```

整个扩展运行在浏览器内部，没有单独的后端服务。GTD 数据、界面偏好等保存在 `chrome.storage.local`。
